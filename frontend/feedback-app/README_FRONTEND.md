## Setup

### Prerequisites

- Node.js 16+
- AWS CDK deployed infrastructure (see `/infrastructure` folder)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables:
   - Copy `.env` and update with your deployed API Gateway endpoint
   - Get the endpoint URL from CDK output: `FeedbackAppApiEndpoint`

3. Start the development server:
```bash
npm start
```

## Usage

### Testing Lambda Functions

1. **Sign In**: Use the Cognito authentication form
2. **Submit Feedback**:
   - Enter feedback text in the textarea
   - Click "Submit Feedback"
   - Note the returned `feedbackId`
3. **Get Recommendation**:
   - Enter the `feedbackId` from step 2
   - Click "Get Recommendation"
   - View the AI-generated recommendation

### API Endpoints Tested

- **POST /feedback**
  - Body: `{ "text": "your feedback" }`
  - Response: `{ "feedbackId": "...", "message": "..." }`

- **GET /recommendation?feedbackId=xxx**
  - Response: Recommendation object with feedback details

## Configuration

### Environment Variables

Create a `.env` file in the root directory:

```env
# API Gateway endpoint (from CDK deployment)
REACT_APP_API_ENDPOINT=https://your-api-id.execute-api.region.amazonaws.com/prod

# AWS Region
REACT_APP_AWS_REGION=eu-central-1
```

### AWS Configuration

The app uses `aws-exports.js` for Cognito configuration, which should be automatically generated by Amplify CLI.

## Troubleshooting

### Common Issues

1. **API Endpoint Not Configured**
   - Update `REACT_APP_API_ENDPOINT` in `.env`
   - Get the URL from CDK deployment output

2. **Authentication Errors**
   - Ensure Cognito User Pool is properly configured
   - Check `aws-exports.js` has correct values

3. **CORS Errors**
   - API Gateway may need CORS configuration
   - Add CORS headers in CDK if needed

4. **Lambda Function Errors**
   - Check CloudWatch logs for Lambda functions
   - Verify IAM permissions are correct

### Debug Mode

Enable debug logging in browser console:
```javascript
localStorage.setItem('debug', 'amplify:*');
```
